# ServerWatch
Система мониторинга работоспособности сервисов, находящихся в продакшене


### Технологический стек и роль каждого компонента

| Компонент  | Роль в проекте                                                                                       |
| ---------- | ---------------------------------------------------------------------------------------------------- |
| React      | Фронтенд: список мониторингов и графики доступности                                                  |
| Node.js    | Бэкенд: API для управления мониторингами, фоновый сервис проверки                                    |
| PostgreSQL | Хранение данных: пользователи, списки сервисов, статусы                                              |
| Redis      | Кэширование последних проверок для быстрой отрисовки статуса, пользовательские сессии, очередь задач |
| Docker     | Контейнеризация для единообразной среды разработки и развёртывания                                   |
| Prometheus | Сбор метрик: среднее время ответа сервисов, сбор ошибок сети и общей нагрузки                        |
| Grafana    | Визуализация метрик: графики нагрузки, статистика сбоев, uptime сервисов                             |
